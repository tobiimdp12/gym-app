## Run Locally

Clone the project

```bash
  git clone https://github.com/tobiimdp12/gym-app.git
```

Go to the project directory

```bash
  cd gym-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
